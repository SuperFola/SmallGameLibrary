# SmallGameLibrary

A C++17 small game library relying on SFML 2.5.x.

# Dependencies

* CMake >= 3.10
* C++17 compiler
* SFML >= 2.5

# Build instructions

```shell
~/SGL$ cmake . -Bbuild -DCMAKE_BUILD_TYPE=Release
~/SGL$ cmake --build build --config Release
```
